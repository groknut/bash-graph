

## Обработка графа на bash

**Реализовано:**
1. Чтение графа из файла
2. Вывод информации о графе
3. Алгоритм BFS
4. Алгоритм DFS (не факт)

**Запуск программы:**
```bash
./main.sh <file> [oriented]
```

**Для примера**
```bash
./main.sh 1.txt 0
```
Здесь `0` означает, что мы подаем неориентированный граф, `1` - ориентированный граф

**Формат файла**
```txt
A B 5
C D 3
X Y 7
```

**Если возникли ошибки:**
```bash
sed -i 's/\r$//' *.sh
```
